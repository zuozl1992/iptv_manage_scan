# TVMC 快速开始指南

本指南将帮助您快速上手 TVMC，完成从配置到扫描再到频道管理的完整工作流程。

## 📋 工作流程概览

```
软件配置 → 生成扫描文件 → 开始扫描 → 保存结果 → 导入结果 → 频道编辑 → 重复
```

## 🚀 快速开始

### 第一步：软件配置

1. 启动 TVMC，切换到 **「软件配置」** 标签页
2. 在 **「服务器配置」** 分组中填入：
   - **单播服务地址**：您的流媒体服务器地址（如 `http://192.168.1.1:12345`）
   - **Logo服务地址**：台标图片服务器地址（可选）
   - **协议**：选择 `udp` 或 `rtp`
3. 在 **「生成扫描文件」** 分组中配置：
   - **组播地址**：组播地址模板（如 `239.49.{0#1#2}`）
     - 例如江苏电信采用`239.49.{0#1#8}`，会自动匹配这三个IP段
   - **端口**：端口号（如 `6000#8000#8008`）

> 💡 **地址模板语法**：
> - 零散值：`1#3` 表示 1 和 3
> - 范围值：`[8-10]` 表示 8、9、10
> - 混合使用：`1#3#[5-7]` 表示 1、3、5、6、7

### 第二步：生成扫描文件

1. 在 **「软件配置」** 标签页的 **「生成扫描文件」** 分组中
2. 点击 **「生成扫描文件」** 按钮
3. 选择保存位置和文件名
4. 生成成功后会弹出提示，点击 **「是」** 自动跳转到扫描页面

### 第三步：开始扫描

1. 切换到 **「频道源扫描」** 标签页（如已自动跳转则跳过此步）
2. 确认 **「文件」** 单选按钮已选中，且文件路径已填入
3. 调整扫描参数：
   - **线程数**：建议 8-16（根据服务器性能调整）
   - **超时(ms)**：建议 500-10000（自己搭建的组播服务建议采用10000代表不跳过）
   - **自动跳过**：勾选后，发现相同IP已有成功地址会自动跳过
4. 点击 **「开始」** 按钮开始扫描
5. 等待扫描完成，观察进度条和成功/失败统计

### 第四步：保存扫描结果

1. 扫描完成后，点击 **「保存」** 按钮
2. 选择保存位置和文件名（使用 `.mc` 格式）
3. 扫描结果将保存为可导入的格式

### 第五步：导入扫描结果

1. 切换到 **「软件配置」** 标签页
2. 在 **「导入文件」** 分组中，点击 **「选择文件」** 按钮
3. 选择刚才保存的扫描结果文件
4. 点击 **「导入」** 按钮
5. 导入成功后会显示导入的记录数

### 第六步：频道编辑

1. 切换到 **「频道编辑」** 标签页
2. 查看导入的频道列表
3. 可以编辑以下字段：
   - **频道ID**：EPG指南中的频道ID
   - **频道**：频道名称
   - **分组**：频道分组（如央视、卫视等）
   - **城市**：频道所属城市
   - **描述**：频道描述信息
   - **备注**：额外备注信息
   - **LOGO**：台标文件名
4. 编辑后按回车或点击其他位置自动保存

### 第七步：重复扫描

1. 返回 **「软件配置」** 标签页
3. 重复 **第二步** 到 **第六步**

## 📺 其他功能

### 频道源浏览

切换到 **「频道源浏览」** 标签页，可以：
- 查看所有频道和信号源信息
- 点击表头排序
- 分页浏览（每页50条）

### 频道源检查

切换到 **「频道源检查」** 标签页，可以：
- 检测信号源质量
- 查看视频预览画面
- 对比检测值与数据库值
- 提交检测结果更新

### 导出功能

在 **「软件配置」** 标签页的 **「导出文件」** 分组中，可以导出为：
- **M3U**：标准播放列表格式
- **TXT**：文本格式播放列表
- **XLS**：Excel表格格式
- **CSV**：逗号分隔值格式

### 导入其他格式

除了扫描结果（`.mc` 格式），还支持导入：
- **.m3u**：标准M3U播放列表
- **.txt**：TXT播放列表

## ⚙️ 配置说明

### 地址模板示例

| 模板 | 展开结果 |
|------|----------|
| `239.49.0.1` | 239.49.0.1 |
| `239.49.{0#1#2}` | 239.49.0, 239.49.1, 239.49.2 |
| `239.49.{[0-2]}` | 239.49.0, 239.49.1, 239.49.2 |
| `239.49.{0#1}.[1-10]` | 239.49.0.1-10, 239.49.1.1-10 |
| `{6000#8000}` | 6000, 8000 |
| `{[6000-6010]}` | 6000, 6001, ..., 6010 |

### 扫描参数建议

| 参数 | 建议值 | 说明 |
|------|--------|------|
| 线程数 | 8-16 | 根据服务器最大连接数设置 |
| 超时(ms) | 500-10000 | 局域网500-1000，组播建议使用10000 |
| 自动跳过 | 开启 | 提高扫描效率 |
| 精细扫描 | 关闭 | 仅在需要精确探测时开启 |

## 🔧 故障排除

### 扫描失败率高

- 检查网络连接
- 增加超时时间
- 减少线程数
- 检查地址模板是否正确

### 导入失败

- 检查文件格式是否正确
- 确认文件编码为 UTF-8
- 查看错误提示信息

### 视频预览无画面

- 检查 FFmpeg 是否正确安装
- 确认流媒体地址可访问
- 检查服务器是否在线
